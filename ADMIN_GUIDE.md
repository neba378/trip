# Admin Guide - SAVANNA Ethiopia Travel

## 🔐 Accessing Admin Panel

1. Go to: **http://localhost:5173/admin** (or your-domain.com/admin)
2. Enter your admin **email and password** (ask the owner to create an account for you)
3. Click "Login"

## 📊 Dashboard Overview

When you login, you'll see:

### Statistics Cards (Top)
- **Total Bookings** - All bookings ever made
- **Pending** - Waiting for your verification (🟡 Yellow)
- **Confirmed** - Verified and approved (🟢 Green)
- **Rejected** - Declined bookings (🔴 Red)
- **Revenue** - Total confirmed payments in ETB

### Filter & Search
- **Search Bar** - Find by name, phone, or confirmation code
- **Filter Buttons** - Show All / Pending / Confirmed / Rejected

## ✅ How to Verify a Booking

### Step 1: Find Pending Booking
- Look for bookings with 🟡 **yellow "pending"** badge
- These need your attention

### Step 2: Review Details
Each booking card shows:
- **Customer Name** - Who booked
- **Phone & Email** - Contact info (click to call/email)
- **Trip Details** - Which trip, date, number of people
- **Amount** - Total price in ETB or USD
- **Confirmation Code** - Unique booking ID
- **Notes** - Any special requests

### Step 3: Check Payment Screenshot
- Click **"View Payment Screenshot"**
- Screenshot will appear showing their payment confirmation
- **Verify carefully:**
  - ✅ Amount matches the booking total **EXACTLY**
  - ✅ Payment went to **correct account** (Telebirr/CBE/Abyssinia)
  - ✅ Screenshot shows **"Transaction Successful"** or similar
  - ✅ Date is **recent** (within last few days)
  - ✅ Transaction ID or reference number is visible
  - ✅ Screenshot looks **genuine** (not edited)
  - ✅ Shows the **full payment confirmation screen**

**What a GOOD screenshot looks like:**
- Clear and readable
- Shows success message
- Amount is visible and correct
- Date/time stamp visible
- Transaction reference number shown
- Looks authentic (not photoshopped)

### Step 4: Confirm or Reject

**If payment is valid:**
1. Click **"Confirm Payment"** (green button)
2. Status changes to "confirmed" 🟢
3. Call customer to confirm their spot: Use phone number shown
4. Tell them their booking is confirmed

**If payment is invalid/fake:**
1. Click **"Reject"** (red button)
2. Status changes to "rejected" 🔴
3. Call customer to explain the issue

## 🔍 Search & Filter Tips

### Search Examples:
- Type **"Abebe"** - Find all bookings by Abebe
- Type **"0911"** - Find bookings with that phone number
- Type **"SAV-ABC123"** - Find specific confirmation code

### Filter Usage:
- **All** - See everything
- **Pending** - Only unverified bookings (your to-do list)
- **Confirmed** - Successfully verified bookings
- **Rejected** - Declined bookings

## 💡 Best Practices

### Daily Routine:
1. **Morning** - Login and check pending bookings
2. **Verify Payments** - Process all pending within 24 hours
3. **Call Customers** - Confirm all verified bookings
4. **Evening** - Final check for new bookings

### Payment Verification Checklist:
- [ ] Amount matches **exactly** (not more, not less)
- [ ] Correct account (Telebirr: 0944780840 / CBE: 1000524300657 / Abyssinia: 67531167)
- [ ] Screenshot is **clear and readable**
- [ ] Shows **"Transaction Successful"** or confirmation message
- [ ] Date is **recent** (within last 2-3 days of booking)
- [ ] Transaction ID/reference number **visible**
- [ ] Screenshot looks **authentic** (not edited/photoshopped)
- [ ] Shows **full confirmation screen** (not cropped suspiciously)

### Red Flags (Reject These):
- ❌ Screenshot is **blurry/unreadable**
- ❌ Amount **doesn't match** booking total
- ❌ Payment to **wrong account** or wrong number
- ❌ **Old screenshot** (from weeks/months ago)
- ❌ Screenshot looks **edited/photoshopped** (check for inconsistencies)
- ❌ Screenshot from **Google images** or internet
- ❌ Shows **"Transaction Failed"** or pending status
- ❌ **Cropped** in suspicious way (hiding important details)
- ❌ Different **currency** than booking (ETB vs USD mismatch)
- ❌ Screenshot of someone else's payment (different name/number)

## 🔒 Security

### Keep Password Safe:
- Don't share admin password
- Change it regularly (edit code if needed)
- Logout when done

### Logout:
- Click **"Logout"** button (top right)
- Always logout on shared computers

## 📞 Need Help?
